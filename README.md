# Student Performance Analytics Dashboard

An interactive Machine Learning dashboard built using Python, Streamlit, and Scikit-learn for analyzing and predicting student performance.

## Project Overview

This application allows users to upload a student performance dataset, explore it through interactive visualizations, and generate predictions using a trained Random Forest Regression model.

#Live Demo
https://student-performance-dashboard-497bqqfyv2gbotywkvzgaw.streamlit.app/

## Features

- Upload CSV datasets
- Interactive dashboard
- Exploratory Data Analysis (EDA)
- Data visualization
- Random Forest Regression model
- Performance prediction
- Download prediction results as CSV

## Installation

Clone the repository:

```bash
git https://github.com/meerab1500/student-performance-dashboard

Move into the project directory:

```bash
cd Student-Performance-Dashboard
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
